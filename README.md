# IIT Roorkee Mathematics MA Mock Test

🎓 **Free online mock test for IIT Roorkee Master of Arts (Mathematics) entrance examination**

## ✨ Features

- **65 Questions** - 10 General Aptitude + 55 Mathematics Questions  
- **3 Hour Timer** - Auto-submit on timeout  
- **Live Scoring** - Instant results with negative marking  
- **Question Navigator** - Easy navigation between questions  
- **Mark for Review** - Mark difficult questions  
- **Responsive Design** - Works on all devices  

## 📊 Question Distribution

| Section | Questions | Marks |
|---------|-----------|-------|
| General Aptitude | 1-10 | 1 mark each |
| Mathematics I | 11-35 | 1 mark each |
| Mathematics II | 36-65 | 2 marks each |

## ⚙️ Scoring Rules

- ✅ **Correct Answer:** Full marks  
- ❌ **Wrong Answer:** -1/3 mark (1-mark Q), -2/3 mark (2-mark Q)  
- ⏭️ **Unattempted:** 0 marks  

## 🚀 How to Use

1. **Start the Test:** Click [here](https://ysmallapur21-star.github.io/iit-roorkee-mock-test/)
2. **Select Answers:** Click on options to select
3. **Navigate:** Use Previous/Next buttons or click question numbers
4. **Mark for Review:** Click the mark icon for difficult questions
5. **Submit:** Click Submit to see results
